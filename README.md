# OreUI 设计

### 基于原生HTML无需任何插件实现OreUI界面组件

## 何为OreUI?
OreUI是Mojang工作室使用网络技术用于开发用户界面(UI)的一种开放资源构建模块.

你可以从 https://github.com/Mojang/ore-ui 了解更多.

## 仓库介绍
本仓库主要通过原生HTML实现OreUI的内容组件,无需借助任何插件.

以下项目运用了该技术
